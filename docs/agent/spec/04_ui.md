## 4. ページ仕様

---

### 4-1. ホーム（`/`）

**構成セクション（上から順）**

#### ① ヒーローセクション
- 背景: 六角形グリッド + カラーオーブ（パープル・グリーン・レッド）
- 左側: バッジ「EDUCATIONAL TOXICOLOGY PLATFORM」+ タイトル「毒学」+ キャッチコピー + CTAボタン
- 右側: SVG 分子構造アニメーション（回転リング + 六員環 + 元素ラベル）
- CTAボタン: 「🔬 学習を始める → /categories」「📖 毒図鑑を見る → /toxins」

#### ② 統計バー
| 数値 | ラベル |
|------|--------|
| 480+ | 収録毒物種 |
| 120+ | 化学構造解説 |
| 60+ | 歴史・文化記事 |
| 4 | 学習カテゴリ |

#### ③ カテゴリセクション（`#categories`）
- 見出し: `// CATEGORIES` / 毒を学ぶ4つの入口
- 2×2グリッドでカテゴリカードを表示
- 各カードには: アイコン / 英語名 / 日本語名 / 説明文 / タグチップ / 種数 / 矢印アイコン
- ホバー時: `translateY(-4px)` + カテゴリカラーのグロー

#### ④ スポットライトセクション（`#spotlight`）
- 見出し: `// SPOTLIGHT` / 注目の毒
- 3列グリッドでスポットライト毒物カード表示（API: `toxins.spotlight`）
- 各カードには: 絵文字ビジュアル / カテゴリバッジ / 英語名 / 日本語名 / 説明文 / 危険度バッジ / 「詳しく読む →」

#### ⑤ 化学セクション（`#chemistry`）
- 見出し: `// CHEMISTRY` / 毒の化学構造
- 2列グリッド
  - 左: 分子式カード（化学式 / 分子量 / LD₅₀ / 毒素クラス / 作用標的 / 産生生物 / 解毒剤）
  - 右: 作用機序カード（番号付きステップリスト）

#### ⑥ 歴史タイムラインセクション（`#history`）
- 見出し: `// HISTORY & CULTURE` / 毒と人類の歴史
- 縦タイムライン（6件サンプル表示）
- 「もっと見る → /history」リンク
- 各アイテム: 年代（琥珀色）/ タグ（culture/science/incident/medicine）/ タイトル / 説明文

#### ⑦ 最新記事セクション（`#articles`）
- 見出し: `// LATEST ARTICLES` / 最新記事
- 3×2グリッドで最新6件表示（API: `articles.latest`）
- 各カード: 絵文字 / カテゴリ英語 / タイトル / 抜粋 / 公開日 / カテゴリタグ

#### ⑧ CTAバナー
- テキスト: 「毒図鑑を探索する」「480種以上の毒物データベースを...」
- ボタン: 「🔍 図鑑を開く → /toxins」「📊 危険度ランキング → /ranking」

### 4-2. 毒図鑑一覧（`/toxins`）

**レイアウト**
- サイドバー（左240px）+ メインコンテンツ（右側）

**サイドバー**
- カテゴリフィルタ（ラジオボタン）: すべて / 動物の毒 / 植物の毒 / 毒の化学 / 歴史と文化
- 危険度フィルタ（チェックボックス）: ●〜●●●●●
- タグフィルタ（タグチップクリック）

**メインコンテンツ**
- ページタイトル: 「毒図鑑」+ 件数表示
- 検索ボックス（インライン）
- ソート切り替え: 名前順 / 危険度順 / 新着順
- 毒物カードグリッド（3列 → モバイル1列）

**毒物カード（一覧用）**
```
┌──────────────────────────────┐
│  🐡  [ANIMAL TOXIN]          │
│  Tetrodotoxin (TTX)          │
│  テトロドトキシン             │
│  フグをはじめ複数の生物が...   │
│  ●●●●● 極    C₁₁H₁₇N₃O₈    │
│  🐍ヘビ毒  🦂サソリ毒        │
└──────────────────────────────┘
```

- ホバー: `translateY(-4px)` + 上部にカテゴリカラーのライン
- クリック → `/toxins/:slug`

- 1ページ20件 / ページャーナビゲーション

### 4-3. 毒物詳細（`/toxins/:slug`）

**レイアウト**: 2カラム（本文 + サイドバー）

**ヘッダーエリア**
- カテゴリバッジ + 英語名（Space Mono）
- 日本語名（大見出し / Noto Serif JP）
- 危険度バッジ
- タグチップ群
- ブックマークボタン（ログイン時）

**本文左カラム（2/3幅）**

① 説明文セクション
- 説明文（`p.description`）

② 化学情報カード（`// MOLECULAR FORMULA`）
- 分子式表示（大きい Space Mono フォント）
- プロパティグリッド（2×3）:
  - 分子量 / LD₅₀ / 毒素クラス / 作用標的 / 産生生物 / 解毒剤

③ 作用機序カード（`// MECHANISM OF ACTION`）
- 番号付きステップリスト（縦に矢印コネクタ）

**サイドバー右カラム（1/3幅）**
- 関連毒物カード（同カテゴリ / 同タグの毒物 最大4件）

## 5. レスポンシブ対応

| ブレークポイント | グリッド変化 |
|----------------|------------|
| `>= 1280px` (xl) | 3列グリッド（毒物・記事） |
| `>= 768px` (md) | 2列グリッド |
| `< 768px` (sm) | 1列グリッド |

- ナビゲーション: モバイル時はハンバーガーメニュー
- サイドバー: モバイル時は折りたたみアコーディオン

---

## 6. アニメーション

| 要素 | アニメーション |
|------|--------------|
| ヒーロー SVG 分子 | `float` 上下 6秒ループ |
| 分子外リング | `spin-slow` 20秒回転 |
| 分子内リング | `spin-slow` 30秒逆回転 |
| ヒーローバッジの点滅ドット | `pulse` 2秒ループ |
| カードホバー | `translateY(-4px)` 0.3s |
| ナビリンクホバー | 下線スライドイン 0.2s |
