# 全体概要仕様書

## 1. プロジェクト概要

### サービス名
**毒学 ~poison learning~**

### コンセプト
自然界に潜む「毒」を科学・歴史・文化の視点で学ぶ、**教育目的の毒学サイエンス学習プラットフォーム**。  
動物・植物が持つ毒の仕組み、化学構造・作用機序、そして人類と毒が織りなす歴史と文化を深く探求できる。

### 想定ユーザー
- 毒や生物化学に興味を持つ一般学習者
- 学生（理科・生物・化学・歴史）
- サイエンスコンテンツの愛好者

---

## 2. 機能一覧

### 2-1. コア機能（必須）

| 機能 | 説明 |
|------|------|
| **毒図鑑（Toxin Encyclopedia）** | 480種以上の毒物をカテゴリ・危険度・分類で閲覧・検索できるデータベース |
| **カテゴリ別学習** | 動物の毒 / 植物の毒 / 毒の化学 / 歴史と文化 の4カテゴリ |
| **毒物詳細ページ** | 化学式・分子量・LD₅₀・作用機序・産生生物・解毒剤などを詳細表示 |
| **記事コンテンツ** | 各テーマを深掘りした読み物記事の公開・閲覧 |
| **歴史タイムライン** | BC 399〜現代まで、毒にまつわる歴史的出来事をタイムライン表示 |
| **全文検索** | 毒物名・記事タイトル・説明文を横断検索 |
| **危険度ランキング** | LD₅₀値に基づく危険度スコアでランキング表示 |

### 2-2. 拡張機能（追加実装）

| 機能 | 説明 |
|------|------|
| **化学構造一覧** | 分子式・分子量・毒素クラスで絞り込める化学物質一覧 |
| **ユーザー認証** | 会員登録・ログイン（better-auth） |
| **ブックマーク** | ログインユーザーが毒物・記事をお気に入り保存 |
| **関連コンテンツ** | 毒物詳細・記事下部に関連コンテンツを表示 |
| **タグ絞り込み** | タグ（ヘビ毒・神経毒・アルカロイドなど）でコンテンツを横断検索 |

## 4. プロジェクト構成

```
better-t-app/
├── apps/
│   ├── web/         # フロントエンド（TanStack Router + Vite）
│   └── server/      # バックエンド（Hono + Bun）
└── packages/
    ├── api/         # oRPC ルーター定義
    ├── auth/        # better-auth 設定
    ├── config/      # 共通 tsconfig
    ├── db/          # Drizzle スキーマ + マイグレーション
    ├── env/         # 環境変数バリデーション
    └── ui/          # shadcn/ui コンポーネント
```

## 5. ページ構成

| パス | ページ名 | 概要 |
|------|---------|------|
| `/` | ホーム | ヒーロー / 統計 / カテゴリ / スポットライト / 化学 / 歴史 / 記事 |
| `/toxins` | 毒図鑑一覧 | 全毒物一覧（検索・カテゴリ・危険度フィルタ） |
| `/toxins/:slug` | 毒物詳細 | 化学情報・作用機序・タグ・関連毒物 |
| `/categories` | カテゴリ一覧 | 4カテゴリの概要カード |
| `/categories/:slug` | カテゴリ詳細 | カテゴリ内毒物一覧 |
| `/articles` | 記事一覧 | 全記事リスト（タグ・カテゴリフィルタ） |
| `/articles/:slug` | 記事詳細 | 記事本文 + 関連記事 |
| `/chemistry` | 化学構造一覧 | 分子式・LD₅₀ 一覧・絞り込み |
| `/history` | 歴史タイムライン | 年代別タイムライン全表示 |
| `/ranking` | 危険度ランキング | LD₅₀ 値によるランキング |
| `/search` | 検索結果 | 全文検索結果（毒物 / 記事 混在） |
| `/login` | ログイン | サインイン・サインアップ |
| `/dashboard` | マイページ | ブックマーク一覧・プロフィール |

## 7. 非機能要件

| 項目 | 要件 |
|------|------|
| **レスポンシブ対応** | モバイル〜デスクトップ対応（Tailwind breakpoints） |
| **アクセシビリティ** | セマンティックHTML、alt属性、キーボード操作対応 |
| **免責事項** | すべてのページに「教育目的」の注記を表示 |
| **テスト** | 機能追加時にユニットテストを実施 |
| **コード品質** | Biome によるリント・フォーマット統一 |
